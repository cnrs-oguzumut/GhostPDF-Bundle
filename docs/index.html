Privacy Policy

Last updated: December 2025

GhostPDF is designed with privacy as a core principle. We do not collect, store, or transmit any user data.

Data Collection

This app collects no personal information. All PDF compression happens entirely on your device. Your files are never uploaded to any server.

Network Access

This app does not require internet access and makes no network connections. It works completely offline.

Third-Party Services

This app does not integrate with any third-party analytics, advertising, or tracking services.

Common Issues

  • PDF not compressing: Some PDFs are already optimized and may not compress further.
  • Ghostscript Issues (Free Version): If not detected, install via Homebrew: brew install ghostscript. The Pro Bundle (v2.0) includes Ghostscript and requires no setup.
  • Quality too low: Try a higher DPI setting in Pro Mode.
