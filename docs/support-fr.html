Problèmes Courants

  • PDF ne se compresse pas : Certains PDF sont déjà optimisés et ne peuvent pas être compressés davantage.
  • Problèmes Ghostscript (Version Gratuite) : S'il n'est pas détecté, installez-le via Homebrew : brew install ghostscript. Le Pro Bundle (v2.0) inclut Ghostscript et ne nécessite aucune configuration.
  • Qualité trop faible : Essayez un réglage DPI plus élevé en mode Pro.

Configuration Requise

  • Pro Bundle (v2.0) : macOS 13.0+ (Ventura) • Apple Silicon uniquement
  • Version Gratuite (v1.5) : macOS 11.0+ (Big Sur) • Intel et Apple Silicon
